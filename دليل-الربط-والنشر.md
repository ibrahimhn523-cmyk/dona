# دليل ربط ونشر دونا (الوكيل الذكي)

الكود جاهز ومبني بنجاح (`npm run build` نجح). هذه خطوات الربط والنشر. مفتاح Claude API هو **آخر خطوة** كما اتفقنا.

الترتيب: Supabase أولاً (قاعدة البيانات والدخول)، ثم GitHub (رفع الكود)، ثم Vercel (النشر)، ثم مفتاح API.

---

## الخطوة ١ — Supabase (قاعدة البيانات + تسجيل الدخول)

1. أنشئ مشروعاً جديداً على https://supabase.com (أنت من ينشئ الحساب/المشروع).
2. من **SQL Editor** الصق كامل محتوى `supabase/schema.sql` ثم **Run**. سيُنشئ الجداول السبعة والصلاحيات.
3. من **Project Settings → API** انسخ:
   - `Project URL` → سيصبح `NEXT_PUBLIC_SUPABASE_URL`
   - `anon public key` → سيصبح `NEXT_PUBLIC_SUPABASE_ANON_KEY`
4. لإطالة الجلسة (تبقى مستمرة): **Authentication → Settings**:
   - فعّل Refresh token rotation.
   - اجعل JWT expiry = 3600 (ساعة).
   - عطّل/أطل أقصى مدة للخمول (Inactivity timeout) قدر المتاح.

> دوري هنا: أرشدك خطوة بخطوة عبر Claude in Chrome على حسابك. لا أنشئ حساباً ولا أُدخل بياناتك بنفسي.

---

## الخطوة ٢ — GitHub (رفع الكود)

1. أنشئ مستودعاً (repository) جديداً خاصاً.
2. ارفع محتوى مجلد `dona-agent` (دونه `node_modules` و`.next` — مستثناة في `.gitignore`).
3. عبر Claude in Chrome، أستطيع مساعدتك في الرفع بعد أن تأذن لي بالوصول للمستودع.

---

## الخطوة ٣ — Vercel (النشر)

1. على https://vercel.com اربط حساب GitHub واستورد المستودع.
2. Vercel يكتشف Next.js تلقائياً — اترك الإعدادات الافتراضية.
3. **قبل النشر**، في **Environment Variables** أضف (بدون قيم سرّية في الكود):
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
4. انشر (Deploy). ستحصل على رابط `https://...vercel.app`.

---

## الخطوة ٤ — مفتاح Claude API (الأخيرة)

1. من https://console.anthropic.com أنشئ مفتاحاً (API Key).
2. في Vercel → Environment Variables أضف:
   - `ANTHROPIC_API_KEY` = المفتاح
3. أعد النشر (Redeploy) ليأخذ المتغير الجديد.

> المفتاح سرّ: لا يُكتب في أي ملف بالمشروع، ولا يُرسل في المحادثة. أنت من يضعه في Vercel مباشرة.

---

## التشغيل المحلي (اختياري للتجربة قبل النشر)

1. انسخ `.env.example` إلى `.env.local` واملأ القيم الثلاث.
2. `npm install` ثم `npm run dev`، وافتح `http://localhost:3000`.
